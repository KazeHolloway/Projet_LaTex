# Projet_LaTex | Soutenance LP Génie Logiciel

Ce dépôt contient tous les fichiers nécessaires à la génération du fichier PDF de mon document de Soutenance en vue de l'obtention de la Licence Professionnelle Génie Logiciel.
Ce projet est entièrement rédigé en **LaTex**, avec une organisation en plusieurs sections et annexes.

## Contenu du dépôt :
- `main.tex` : fichier principal du document
- `biblio.bib` : base bibliographique
- `section1_preliminaires/` : pages préliminaires (dédicace, remerciements, sommaire…)
- `section2_contenu/` : développement du projet (présentation, analyse, conception, réalisation…)
- `section3_fin/` : annexes et références
- `images/` : illustrations et schémas
- `main.pdf` : version compilée du document (pour consultation rapide)

## Objectif
Ce projet illustre la **conception et réalisation d’une application cross‑plateforme de mise en relation entre prestataires et clients pour services professionnels à CIS‑CONGO.**

## Technologies utilisées
- **LaTeX** (compilation avec TeXstudio)
- **BibTeX** pour la gestion des références
- Packages : `geometry`, `fancyhdr`, `microtype`, `tikz`, etc.
